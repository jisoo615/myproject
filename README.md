# 개인프로젝트 -진행중

진행중인 프로젝트로, 이미 완성이 된 기능들만을 토대로 작성했습니다. 

### rest api와 ajax 방식 게시판

• Java
• Spring Boot
• Spring Security
• Spring Data JPA
• Thymeleaf
• Ajax
• MyBatis

회원가입/로그인 기능, 글작성 crud, 페이징과 검색 기능, spring security로 접근권한 제한과 구글로그인 기능

로그인은 spring security를 사용해 '구글로 로그인 하기' 기능을 완성

페이징 처리와 검색 기능은 mybatis를 사용, 나머지는 jpa 사용
